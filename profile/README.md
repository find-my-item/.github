## ✍️ 프로젝트 개요

- **프로젝트명:** Find-My-Item (FMI)
- **프로젝트 기간:** 2024.09 ~ 진행중
- **목표:** 분실자와 습득자를 연결하는 지도 기반 서비스 구축
- **주요 타겟 사용자:** 분실물 신고자, 습득물 제보자

---

## ✍️ 프로젝트 소개

### 프로젝트 배경

기존 분실물 찾기 절차는 다음과 같은 문제를 가지고 있습니다:

1. **분산된 신고 채널**
   - 경찰청, 공공기관, SNS 등 여러 경로로 분리되어 효율이 낮음
2. **늦은 대응 시간**
   - 실시간 연결이 어려워 신속한 대응이 불가능함
3. **한정된 정보 제공**
   - 텍스트 중심 게시글 구조로 위치 정보 활용이 제한됨

**Find-My-Item**은 이러한 문제를 해결하기 위해  
**지도 기반 위치 등록**, **실시간 채팅**, **공공데이터 포털 API**를 통합한  
**통합형 분실물·습득물 매칭 서비스**입니다.

---

## 🚀 프로젝트 목표

1. **신속한 매칭** — 지도·검색 기반 분실물 탐색
2. **실시간 소통** — 채팅으로 직접 연결
3. **편리한 사용성** — 반응형 UI와 직관적 인터페이스
